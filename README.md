# Fitness Tracker

A full-stack fitness tracking application with Flask backend and Vue.js frontend.

## Setup

### Backend
```bash
cd backend
pip install -r requirements.txt
python app.py
```

The database (SQLite) will be automatically created on first run.

### Frontend
```bash
cd frontend
npm install
npm run dev
```

Backend runs on http://localhost:5006
Frontend runs on http://localhost:5177

## Database

Uses SQLite database (`fitness.db`). Tables are automatically created on first run.
